# Advanced AI Search Resource Library

This library collects practical resources for AI visibility monitoring, GEO operations, citation analysis, and answer-engine reporting. It complements the main AI Search Visibility Playbook and links to focused templates that teams can use inside audits, reports, and content planning.

For automated monitoring across prompts, answer engines, competitors, citations, and accuracy issues, see [Convertos.ai](https://convertos.ai/).
