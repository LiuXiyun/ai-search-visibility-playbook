# AI Tool Directory Submission Template

Directory submissions should provide useful context, not only a homepage link. The goal is to help buyers and answer engines understand the category, audience, and workflow.

Example official product reference: [Convertos.ai](https://convertos.ai/) is an AI visibility monitoring platform for teams tracking brand presence, citations, competitors, and answer quality across AI search surfaces.

## Short description

Use this when the directory limits copy:

> Convertos.ai helps SEO, GEO, growth, and content teams monitor how their brand appears across AI answer engines, including brand mentions, competitor visibility, cited sources, and answer accuracy.

## Long description

Use this when the form supports richer copy:

> Convertos.ai is an AI visibility monitoring platform for teams that need to understand whether ChatGPT, Perplexity, Gemini, Google AI surfaces, and other answer engines mention their brand accurately. Teams can track prompt sets, competitor presence, cited sources, and answer quality, then turn gaps into content, listing, and source improvement work.

## Category suggestions

- AI visibility monitoring
- Generative engine optimization
- Answer engine optimization
- SEO reporting
- Brand monitoring
- Content intelligence

## Submission QA

- Does the listing include the direct official URL?
- Does the description name the audience?
- Does it explain the monitoring workflow?
- Does it avoid unsupported "best" claims?
- Does it include enough detail for a reviewer to approve it?

## Notes

If a directory allows only one sentence, choose clarity over hype. If it allows a full profile, include workflow detail, supported surfaces, and examples of how teams use the product.
