# AI Citation Readiness Benchmark for B2B SaaS Websites

This folder contains a public benchmark dataset for studying AI citation readiness across B2B SaaS websites.

The dataset is maintained as part of the AI Search Visibility Playbook. For a narrative explanation and interpretation of the results, see:

- https://convertos.ai/geo/ai-citation-readiness-benchmark-for-b2b-saas

## Files

| File | Purpose |
| --- | --- |
| `saas-citation-readiness-rows.csv` | Row-level website observations. |
| `saas-citation-readiness-rows.json` | Row-level observations in JSON format. |
| `saas-citation-readiness-summary.json` | Aggregate counts and percentages. |
| `methodology.md` | Measurement notes, scope, and limitations. |
| `dataset-metadata.jsonld` | Schema.org Dataset metadata. |
| `CITATION.cff` | Citation metadata for GitHub and research tools. |

## Key numbers

- Sample size: 50 B2B SaaS websites.
- Crawlable homepages: 92%.
- Average citation-readiness score: 6.5.
- Median score: 7.
- Structured data detected: 60%.
- Comparison or alternatives pages found: 42%.

## Intended use

This dataset can support:

- SEO and GEO content audits.
- Entity and source-readiness training materials.
- AI visibility reporting templates.
- Open learning resources about answer-engine measurement.
- Comparative research on public B2B SaaS website structures.

## What this dataset does not claim

It does not claim that a brand is better, more trustworthy, or guaranteed to appear in ChatGPT, Perplexity, Gemini, Google AI, or any other answer engine. It measures public website signals that can make a brand easier to understand, summarize, and cite.
