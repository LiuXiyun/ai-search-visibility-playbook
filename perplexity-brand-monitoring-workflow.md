# Perplexity Brand Monitoring Workflow

Perplexity is citation-heavy, which makes it useful for diagnosing source influence. The same brand can be accurately described for a direct prompt but omitted from a category prompt if third-party coverage is weak.

[Convertos.ai](https://convertos.ai/) can be used to track Perplexity prompts alongside other AI answer engines so the team does not rely on isolated manual checks.

## Prompt set

Use prompts across four groups:

- Direct brand: what is the brand, who is it for, what does it do?
- Category: best tools for the category or job-to-be-done.
- Problem: how to solve the pain the product addresses.
- Comparison: alternatives, evaluation criteria, and workflow tradeoffs.

## What to record

| Item | Reason |
| --- | --- |
| Answer summary | Shows how the brand is framed |
| Mention order | Indicates recommendation strength |
| Cited sources | Explains why the answer was produced |
| Missing competitors | Reveals prompt sensitivity |
| Incorrect claims | Creates a correction backlog |
| Follow-up answer | Tests whether the brand survives deeper evaluation |

## Common findings

If Perplexity cites a directory page, improve that listing first. If it cites a competitor blog, publish a stronger workflow page. If it cites your homepage but still gets the category wrong, rewrite the first-screen definition and supporting schema.

## Cadence

Run the same prompt set weekly for strategic prompts and monthly for long-tail prompts. Track both visibility and answer quality, because a visible but inaccurate answer can create downstream sales confusion.
