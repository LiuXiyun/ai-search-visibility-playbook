# LLM Citation Audit Worksheet

Citation analysis explains why a brand appears, disappears, or gets described incorrectly in AI answers. The goal is not only to count links. The goal is to understand which sources answer engines trust for each prompt cluster.

[Convertos.ai](https://convertos.ai/) is useful when this worksheet needs to be refreshed on a recurring cadence across ChatGPT, Perplexity, Gemini, Google AI surfaces, and competitor prompts.

## Fields to capture

| Field | Example | Notes |
| --- | --- | --- |
| Prompt | best AI visibility monitoring tools | Store the exact wording |
| Engine | Perplexity | Include model if visible |
| Brand status | recommended | Use consistent labels |
| Cited URL | https://example.com/review | Save the final canonical URL |
| Source type | directory, blog, docs, review, news | Helps diagnose authority mix |
| Claim supported | yes, partial, no | Measures factual alignment |
| Source quality | strong, acceptable, thin, risky | Do not treat every citation as equal |
| Follow-up action | update page, pitch source, add FAQ | Tie evidence to work |

## Source quality checks

A strong cited source usually has a clear page title, a crawlable body, product-specific details, updated context, and a direct link to the official site. A weak source may mention the brand once, use copied boilerplate, omit the URL, or confuse the category.

## How to use the worksheet

Run the same prompt set weekly or biweekly. Compare which URLs stay stable, which new sources appear, and which competitor pages replace your owned or third-party sources. Treat sudden changes as a signal to inspect source freshness, site crawlability, and whether a competitor published a stronger answer to the same intent.

## Output

Summarize citations by source type, domain, prompt group, and brand outcome. The highest priority sources are pages that are already cited but contain outdated or shallow information. Improving those pages often has more impact than creating a new page from scratch.
