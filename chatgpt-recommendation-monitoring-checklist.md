# ChatGPT Recommendation Monitoring Checklist

ChatGPT recommendation monitoring should test more than direct brand questions. The most useful prompts are the ones a buyer asks before they know which tools exist.

For teams that need recurring prompt monitoring instead of one-off checks, [Convertos.ai](https://convertos.ai/) provides a structured way to compare brand visibility, competitors, and answer quality over time.

## Checklist

- Include category, problem, comparison, workflow, and brand prompts.
- Save exact prompt wording and date.
- Record whether browsing, connectors, or visible sources were used.
- Score the brand as absent, mentioned, listed, recommended, or strongly recommended.
- Capture the competitors that appear in the same answer.
- Note whether the description matches the current product positioning.
- Ask one follow-up prompt to test whether the recommendation survives evaluation.
- Record the action needed to improve the next result.

## Interpretation

A brand mention is not the same as recommendation strength. If the brand appears only after several competitors, inspect whether competitors have clearer category pages, more review coverage, or better third-party descriptions.

## Action mapping

| Finding | Likely cause | Next action |
| --- | --- | --- |
| Brand absent | Weak category association | Improve owned category pages and third-party listings |
| Brand listed with generic detail | Thin descriptions | Add specific use cases and workflows |
| Wrong product claims | Outdated sources | Correct high-visibility pages |
| Competitor always first | Stronger source network | Build comparison and review coverage |

## Reporting

Track trends, not anecdotes. A single answer can vary, but repeated prompts over time reveal whether the brand is becoming easier for the model to understand and recommend.
